## Intro
QUASH Quite A Shell is similar to csh, bash or other popular shell programs.

## Install
To build Quash use:
> `make`

To generate Quash documentation
> `make doc`
> open html/index.html for doxygen report

To clean quash use:
> `make clean`

To zip projects
> 'make zip'

## Usage
To run Quash use:
> `./quash`
or
> `make test`
